Приземление
Инструкция:
1. Выделите 3D-сетку и объекты для приземления одновременно
2. Нажмите "Приземлить" для посадки объектов на 3D-сетку

Смещение: "Применить" — просто сдвиг по Z (без 3D-сетки)
Размеры и разметка
Инструкция:
1. Создать разметку:
• Установите шаг разметки (например, 1000 мм)
• Нажмите "Установить" для применения шага
• Выделите объекты для разметки
• Нажмите "Создать разметку"

2. Размеры к линии:
• Выделите объекты (Object/Column/Lamp)
• Нажмите "Размеры к линии"
• Укажите две точки линии

3. Размеры между объектами:
• Выделите объекты (Object/Column/Lamp)
• Нажмите "Размеры между объектами"

4. Размеры к точке:
• Выделите объекты (Object/Column/Lamp)
• Нажмите "Размеры к точке"
• Укажите целевую точку
Создание контуров по линии

1. Нажмите "Выбрать линию" и выберите базовую линию
2. Задайте ширину контура
3. Нажмите кнопку для создания контуров
Примечание: Для spline нужен шаг, для остальных линий - только ширина

Создаёт контуры по базовой линии с заданными параметрами.
Для spline: откладывает точки с заданным шагом, строит перпендикуляры
Для остальных линий: проверяет замкнутость, строит перпендикуляры в начале/конце
Создаёт два боковых контура с замыкающими линиями (для открытых линий).
Внимание: Сначала выберите базовую линию (spline/polyline/дуга/линия).
Массово задать ID выделенным элементам

Переименует ВСЕ выделенные элементы.
Новые ID будут вида "Имя-01", "Имя-02", "Имя-03"...
Слой не меняется.


Введите базовое название и нажмите кнопку. Все выбранные элементы получат ID с номером (например: Сосна обыкновенная-01, Сосна обыкновенная-02...)
Создать папку/слой и перенести элементы

1. Выделите элементы (см. таблицу выше)
2. Укажите путь папки слоёв (например: "Ландшафт/Растения")
3. Укажите название нового слоя
4. Нажмите "Создать и переместить"
ID элементов можно изменить отдельно в блоке выше


Разделяйте подпапки через "/" (например: "Ландшафт/Растения/Деревья")
Что произойдёт:
• Создастся папка для слоёв (если её нет)
• Создастся новый слой в этой папке
• Все выделенные элементы будут перемещены в этот слой
• ID элементов останутся прежними (можно изменить отдельно)
• Это можно отменить через Ctrl+Z
